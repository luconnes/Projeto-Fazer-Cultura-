**Motivação:**

- Nosso projeto é fundamentado na disseminação de conteúdo por meio do formato de streaming. No entanto, também nos inspiramos no canal “TV Cultura”, uma vez que nosso principal objetivo é criar uma plataforma dedicada à divulgação de produções nacionais — sejam elas independentes ou não — como filmes, músicas, animações, entre outros, com o intuito de ampliar o acesso da população à arte. Inicialmente, o foco será voltado para estudantes que desejam publicar seus trabalhos e para artistas independentes. No entanto, nada impede que, futuramente, o projeto possa estabelecer parcerias com órgãos públicos, como a ANCINE, ou até mesmo com instituições privadas.
